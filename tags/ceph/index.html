    什么是Ceph

    Ceph是一个可扩展的，高性能的分布式存储系统。提供了三种不同类型的接口以适应不同的应用场景：

    • block-based: 块存储，可以用做VM的虚拟磁盘
    • object-based: 对象存储，与Amazon S3等常用对象存储兼容
    • file system: POSIX兼容的分布式文件系统，可以被本地系统挂载，并且能被多个客户端共享
